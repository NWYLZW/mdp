# End-to-End

For the primary walkthrough, use [Quick Start](/guide/quick-start). This page is kept as a compact compatibility summary for older links.

## Minimal end-to-end flow

A complete MVP flow is:

1. start server
2. configure your MCP tool to launch the bridge server
3. start client
4. register capabilities
5. open a chat with your agent
6. ask it to list clients or call a tool
7. return result through the server

Once that path works, the protocol has enough shape to support more runtimes.
